# AMSAN BTP

Application de gestion pour petites entreprises du BTP.

## Prérequis

- Node.js v18 ou supérieur

## Installation

```bash
npm install
```

## Lancement

```bash
node backend/server.js
```

Accès : http://localhost:3000

## Structure

```
/amsan-btp/
├── frontend/       Pages HTML (TailwindCSS + Vanilla JS)
├── backend/        API Express + Contrôleurs + Routes
├── database/       Base SQLite (créée automatiquement)
└── package.json
```
